Subject: Seqplot v6.0.0 - Updated Version with VSP Integration

Hi Team,

I'm sharing the updated version of Seqplot v6.0.0 with several new features and improvements.

WHAT'S NEW:
-----------
• VSP API Integration: Automatically fetches and displays existing AAVSO comparison stars
  - Visual overlay with purple rounded boxes and white labels
  - Click any VSP star to see full photometry (V, B-V, V-I, AUID)
  - Automatic magnitude limit calculation (MinMag + 2)

• VSX Enhancements:
  - "Check VSX" button opens star detail page in browser
  - Quick access to full VSX information

• UI Improvements:
  - Help menu with Sequence Team Homepage link
  - Wider Request Star dialog (600px)
  - Better coordinate handling and projection accuracy

INSTALLATION:
-------------
Mac/Linux users:
1. Download: seqplot-v6.0.0-20251108.tar.gz
2. Extract: tar -xzf seqplot-v6.0.0-20251108.tar.gz
3. Run: cd seqplot-v6.0.0-20251108 && ./run.sh

Windows users:
1. Download: seqplot-v6.0.0-20251108.zip
2. Extract the zip file
3. Double-click run.bat (or run from Command Prompt)

REQUIREMENTS:
-------------
- Java 11 or higher (check with: java -version)
- Internet connection for catalog queries

QUICK START:
------------
1. Launch the application
2. Enter a star name (e.g., "EW Cru") or coordinates
3. Click "Find RA & Dec for Star"
4. Select catalogs (APASS9, Gaia, etc.)
5. Click "Get Plot"
6. View in Sky View or Points View
7. Click on purple VSP labels to see existing comparison star data!

Full documentation is included in the package (README.txt and CHANGELOG.txt).
